# ⬡ NexusFiles - Guía Rápida de Uso

## 🎯 Funcionalidades Rápidas

### Duplicar Archivos 
**Atajo:** `Ctrl+D`

1. Selecciona un archivo o carpeta
2. Presiona `Ctrl+D` **O** click derecho → "Duplicar"
3. ¡Listo! Se crea automáticamente con nombre "(copia)"

```
original.txt → original (copia).txt
```

---

### Crear Archivo con Extensión 
**Botón:** `＋ Archivo` en toolbar

1. Haz click en `＋ Archivo`
2. Escribe nombre CON extensión: `script.py`, `index.html`, `documento.pdf`
3. Presiona Enter o click en "Crear"

```
Ejemplos válidos:
- mi_archivo.txt
- script.py
- documento.docx
- config.json
- index.html
```

---

### Cambiar Permisos
**Ubicación:** Menú contextual o Propiedades

#### Opción 1: Desde menú contextual
1. Click derecho en archivo
2. Selecciona `🔒 Solo lectura` o `🔓 Permitir edición`

#### Opción 2: Desde propiedades
1. Click derecho → `ℹ️ Propiedades`
2. En el panel derecho, haz click en `Permitir` o `Bloquear`

```
🔒 Solo lectura = No se puede editar
🔓 Editable = Se puede editar normalmente
```

---

### Comprimir a ZIP 
**Ubicación:** Toolbar o Menú contextual

1. Selecciona archivo(s) o carpeta(s)
2. Click en botón `📦 ZIP` **O** click derecho → "Comprimir (ZIP)"
3. Se crea `nombre.zip` en la misma carpeta

```
archivo.txt → archivo.txt.zip
carpeta/ → carpeta.zip
```

---

### Extraer ZIP 
**Ubicación:** Menú contextual

1. Click derecho en archivo `.zip`
2. Selecciona `📂 Extraer aquí`
3. Se extrae en la carpeta actual

```
documento.zip → documento/ (carpeta)
```

---

### Mostrar/Ocultar Archivos Ocultos 
**Atajo:** `Ctrl+H`

1. Haz click en botón `👁 Ocultos` en toolbar
2. **O** presiona `Ctrl+H`

```
Archivos ocultos = Comienzan con punto (.)
Ejemplos: .config, .gitignore, .env
```

---

### Seleccionar Todo / Deseleccionar
**Atajos:** `Ctrl+A` / `Esc`

#### Seleccionar todo:
1. Presiona `Ctrl+A` **O** click en `✔ Todo`
2. Se resaltan todos los archivos

#### Deseleccionar:
1. Presiona `Esc` **O** click en `✕ Deseleccionar`
2. Se limpian las selecciones

## 📌 Operaciones Básicas

### Navegar
- **Doble clic** en carpeta = Entrar
- **Botón ⬆** = Subir nivel
- **Botones ← →** = Historial
- **Click en breadcrumb** = Ir a esa carpeta

### Crear
- `＋ Carpeta` = Nueva carpeta
- `＋ Archivo` = Nuevo archivo

### Editar
- **F2** = Renombrar
- **Click derecho** = Menú opciones

### Copiar/Mover
- **Ctrl+C** = Copiar
- **Ctrl+X** = Cortar
- **Ctrl+V** = Pegar

### Eliminar
- **Delete** o click derecho → Eliminar
- Va a la papelera (no es definitivo)

---

## 🎨 Vista y Búsqueda

### Cambiar vista
- **⊞** = Vista en cuadrícula (grid)
- **☰** = Vista en lista (tabla)

### Búsqueda
- Escribe en `🔍 Buscar en esta carpeta`
- Se filtra en tiempo real
- Click en **×** para limpiar

### Ordenar
1. Click en encabezado (vista lista)
2. **O** usa el menú `Ordenar por:`
   - Nombre
   - Modificado
   - Tipo
   - Tamaño
3. Click en **↑/↓** para invertir orden

---

## 💡 Consejos

### Para Archivos Grandes
- Use ZIP para reducir tamaño
- Comprimir carpetas antes de compartir
- Extraer solo lo que necesites

### Para Gestión
- Use "Seleccionar Todo" para operaciones masivas
- Los archivos ocultos (Ctrl+H) son para configuración
- Los permisos protegen archivos importantes

### Para Velocidad
- Aprende los atajos (Ctrl+D, Ctrl+H, etc.)
- Menú contextual es más rápido
- Búsqueda es más rápida que navegar

## ❓ FAQ

**P: ¿Puedo recuperar un archivo eliminado?**
A: Sí, va a la papelera de Windows, no se borra definitivamente.

**P: ¿Puedo duplicar una carpeta?**
A: Sí, funciona con carpetas y todo su contenido.

**P: ¿Qué pasa si cambio a "Solo lectura"?**
A: No puedes editar, mover o eliminar ese archivo. Debes cambiar permisos de nuevo.

**P: ¿Puedo comprimir múltiples archivos?**
A: Sí, selecciona varios y haz click en `📦 ZIP`. Se crea un ZIP para cada uno.

**P: ¿Qué formatos ZIP soporta?**
A: Soporta extraer cualquier ZIP válido. La compresión crea ZIP estándar.

**P: ¿Los archivos ocultos son peligrosos?**
A: No, son normales. Sistema y programas usan puntos para archivos de config.

## 🚀 Combos Útiles

### Respaldar una carpeta
1. Selecciona carpeta
2. `Ctrl+D` para duplicar
3. Renombra la copia con fecha
4. ¡Listo!

### Compartir archivos comprimidos
1. Selecciona archivos
2. `📦 ZIP` para comprimir
3. Envía el .zip
4. Destinatario usa `Extraer aquí`

### Organizar archivos rápido
1. `Ctrl+A` para seleccionar todo
2. `Ctrl+X` para cortar
3. Navega a carpeta destino
4. `Ctrl+V` para pegar

### Buscar y duplicar
1. Busca archivo con `🔍`
2. Aparece en resultados
3. `Ctrl+D` para duplicar
4. Modifica la copia

## 🔧 Soporte Técnico

**Si algo no funciona:**

1. Intenta **F5** (Actualizar)
2. Verifica que tengas permisos
3. Comprueba espacio en disco
4. Reinicia la aplicación

**Errores comunes:**

- "No se puede acceder" → Sin permisos (Ctrl+D para cambiar)
- "ZIP no válido" → Archivo corrupto, descarga de nuevo
- "Archivo en uso" → Ciérralo en otra aplicación primero
